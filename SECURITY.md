# Security Policy

## Supported versions

The current stable release (0.1.x) receives security updates.

## Reporting a vulnerability

Email security reports to security@simonvillani.dev. Include a detailed description, steps to reproduce, and any proof-of-concept code. Expect an initial response within 5 business days.

## Disclosure process

1. We will investigate and provide status updates at least once a week.
2. Once a fix is available, we will coordinate a disclosure date with the reporter.
3. Credit will be given in release notes unless anonymity is requested.
